# Stop-EZ

Stop-EZ is an iOS application designed to help non-verbal individuals communicate
basic commands quickly and effectively through a simple, accessible interface.

## Features
- One-tap communication buttons
- Clean, distraction-free UI
- Designed with accessibility in mind

## Architecture
- MVC architecture
- SwiftUI application
- Clear separation between UI and application logic

## Tech Stack
- SwiftUI
- Xcode
- App Store Connect

## Impact
- 1,100+ downloads worldwide
- Used by individuals with special needs to communicate more effectively
